Follow instructions below to compile tfn_lib for Windows, on Visual Studio:

1. Set build configuration to Release (X.Y).
2. Set build platform to x86 or x64.
2. Build Solution.
3. Copy .../vs/[Win32/x64]/Release (X.Y)/tfn_lib/tfn_lib.so
   to the appropriate binary folder:
   /bin/[win32/win64]/[X.Y]/

Follow instructions below to compile tfn_lib for Mac OS X, on xCode:

1. Set active scheme to "Ruby (X.Y) - [32/64]"
2. Execute (Menu) Product > Archive
3. Export the built archive to your documents.
4. Locate tfn_lib.bundle within the exported archive.
5. Copy tfn_lib.bundle to the appropriate binary folder:
   /bin/[osx32/osx64]/[X.Y]/
